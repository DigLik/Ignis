# Документация компонента: Window

Класс `Ignis.Platform.Windowing.Window` представляет собой кроссплатформенную (сфокусированную на Windows/Win32) абстракцию системного окна. Он инкапсулирует создание окна, обработку его системных сообщений, управление жизненным циклом и предоставляет доступ к нативным дескрипторам.

---

## Основные свойства

*   **`Handle`** (`nint`): Системный дескриптор окна (`HWND` в операционной системе Windows). Полезен для интеграции с графическими API (Vulkan, DirectX) или внешними библиотеками.
*   **`Title`** (`string`): Заголовок окна. Доступен для чтения и записи.
*   **`Size`** (`Vector2Int`): Размер клиентской области окна в пикселях. Изменение этого свойства приводит к изменению размеров окна.
*   **`Position`** (`Vector2Int`): Положение левого верхнего угла клиентской области окна на экране в пикселях.
*   **`IsVisible`** (`bool`): Управляет видимостью окна (скрывает или показывает его).
*   **`ShouldClose`** (`bool`): Возвращает `true`, если окно получило сигнал на закрытие (например, от операционной системы или при вызове метода `Close()`).
*   **`CursorVisible`** (`bool`): Управляет видимостью курсора мыши при его нахождении над клиентской областью окна.
*   **`CursorMode`** (`CursorMode`): Задает режим поведения курсора в окне. Доступные режимы:
    *   `CursorFree` — курсор виден и свободно перемещается.
    *   `CursorLocked` — курсор скрыт и заблокирован в центре окна. Этот режим используется для неограниченного относительного движения мыши (например, в шутерах от первого лица).
    *   `CursorGrabbed` — курсор скрыт, но его движение ограничено границами окна.
*   **`IsFullscreen`** (`bool`): Указывает, открыто ли окно во весь экран.

---

## События

*   **`Closed`**: Вызывается при закрытии окна.
*   **`SizeChanged`**: Вызывается при изменении размера клиентской области окна. Предоставляет новые размеры через аргумент `SizeChangedEventArgs.Size`.
*   **`InSizeMove`** (`bool`): Показывает, перетаскивает ли пользователь окно или изменяет его размеры в данный момент.

---

## Методы и жизненный цикл

### Создание окна
Создание экземпляра `Window` автоматически инициализирует нативное окно и выводит его на экран:

```csharp
using Ignis.Platform.Windowing;

// Создание окна
using var window = new Window 
{ 
    Title = "Моя игра", 
    Size = new Vector2Int(1280, 720) 
};
```

### Запуск цикла сообщений (`WaitClose`)
Метод `WaitClose` блокирует вызывающий поток и запускает цикл обработки сообщений операционной системы. Окно продолжает работать, пока свойство `ShouldClose` не станет равным `true`.

```csharp
window.WaitClose();
```

### Закрытие окна (`Close`)
Инициирует закрытие окна программно. Устанавливает флаг `ShouldClose` в значение `true` и отправляет системное сообщение на уничтожение окна.

```csharp
window.Close();
```

### Освобождение ресурсов (`Dispose`)
Класс реализует интерфейс `IDisposable`. При вызове метода `Dispose()` окно уничтожается, а все занятые системные ресурсы освобождаются.

## Пример использования

```csharp
using System;
using Ignis.Core.Numerics;
using Ignis.Platform.Windowing;

class Program
{
    static void Main()
    {
        using var window = new Window
        {
            Title = "Пример использования Window",
            Size = new Vector2Int(800, 600)
        };

        // Подписка на изменение размера
        window.SizeChanged += (sender, e) =>
        {
            Console.WriteLine($"Размер окна изменен на: {e.Size.X}x{e.Size.Y}");
        };

        // Подписка на закрытие
        window.Closed += (sender, e) =>
        {
            Console.WriteLine("Окно было закрыто.");
        };

        // Блокировка и ожидание закрытия
        window.WaitClose();
    }
}
```
